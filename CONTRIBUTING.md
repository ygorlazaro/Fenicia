## How to Contribute

1. **Fork the repository** and create your branch from `develop`.
2. **Describe your changes** clearly in your pull request (PR).
3. **Write tests** for new features or bug fixes when possible.
4. **Follow the code style** and best practices used in the project.
5. **Keep PRs focused**: one feature or fix per PR.
6. **Be respectful** and follow our [Code of Conduct](CODE_OF_CONDUCT.md).

## Reporting Issues

- Use [GitHub Issues](../../issues) to report bugs, suggest features, or ask questions.
- Please provide as much detail as possible.

## Community

- Join discussions, share feedback, and help others.
- All interactions are subject to our Code of Conduct.
